🌟 Application JSP – Guide d’installation (Version Stylée)
Bienvenue sur l’application officielle des Sélections JSP 🔥
Cette app permet de saisir les résultats, consulter les barèmes, suivre le classement et travailler en équipe comme des pros.

🔗 Accès à l’application
Ouvrez ce lien sur votre téléphone :

👉 https://ludo32100-jsp.github.io/jsp-selections/  
Accéder à l’app

🍏 Installation sur iPhone (Safari obligatoire)
Ouvrir le lien dans Safari

Appuyer sur le bouton Partager (📤)

Sélectionner Ajouter à l’écran d’accueil

Appuyer sur Ajouter

L’icône apparaît sur l’écran d’accueil → l’app s’ouvre en plein écran ✨

⚠️ Ne fonctionne pas avec Chrome ou Firefox sur iPhone.

🤖 Installation sur Android (Chrome recommandé)
Ouvrir le lien dans Chrome

Appuyer sur Installer l’application  
(ou Menu ⋮ → Ajouter à l’écran d’accueil)

Valider

L’icône apparaît sur l’écran d’accueil → l’app fonctionne comme une vraie application 🚀
